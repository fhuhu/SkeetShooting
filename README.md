# SkeetShooting
Tir aux pigeons : épouvantail de balcons

## Objectif
Détecter la présence d'un oiseau sur le balcon via une webcam.
Viser l'oiseau.
Tirer avec un pistolet à eau sur la bête.

## Installation

```console
fhh@natacha ~ $ git clone git@github.com:fhuhu/SkeetShooting.git tmp/SkeetShooting
fhh@natacha ~ $ pushd tmp/SkeetShooting
fhh@natacha ~/tmp/SkeetShooting $ python -m venv venv
fhh@natacha ~/tmp/SkeetShooting $ source venv/bin/activate
(venv) fhh@natacha ~/tmp/SkeetShooting $ pip install --upgrade pip
(venv) fhh@natacha ~/tmp/SkeetShooting $ pip install -r requirements.txt
(venv) fhh@natacha ~/tmp/SkeetShooting $ python shooter.py
```
