# SkeetShooting
Tir aux pigeons : épouvantail de balcons

## Objectif
Détecter la présence d'un oiseau sur le balcon via une webcam.
Viser l'oiseau.
Tirer avec un pistolet à eau sur la bête.

## Installation

```console
fhh@natacha ~ $ git clone git@github.com:fhuhu/SkeetShooting.git tmp/SkeetShooting
fhh@natacha ~ $ python -m venv tmp/venv
fhh@natacha ~ $ source tmp/venv/bin/activate
(venv) fhh@natacha ~ $ pip install --upgrade pip
(venv) fhh@natacha ~ $ pip install -r tmp/SkeetShooting/requirements.txt
(venv) fhh@natacha ~ $ pushd tmp/
(venv) fhh@natacha ~/tmp $ python SkeetShooting/shooter.py

```
